RoboBuddy — AI Study Assistant
================================

A single-page AI study assistant that connects to Anthropic's Claude API
(claude-haiku-4-5-20251001) to help students with any subject.

Files
-----
index.html        Main front-end — all UI, styles, and client-side logic in one file.
server.js         Lightweight Node.js HTTP server (port 3000) that serves the HTML
                  and proxies API requests to Anthropic (legacy path; the app now calls
                  the Anthropic API directly from the browser).
package.json      Minimal config — sets type "module" and defines `npm start`.
wrangler.json     Cloudflare Workers/Pages deployment config (assets directory: ".").
build-report.json Build metadata: file inventory, CDN dependencies, API config, and
                  resolved issue log.

Features
--------
- Live chat with RoboBuddy (streaming responses)
- File upload (PDF / DOCX) to use your own notes as context
- Flashcard generator from uploaded notes or a typed topic
- API key entered in-browser, stored in localStorage only
- Direct browser-to-Anthropic API calls (anthropic-dangerous-direct-browser-access header)

Usage
-----
Open index.html directly in a browser, enter a valid Anthropic API key
(sk-ant-...) from console.anthropic.com, and start chatting.

Alternatively, run the Node server for local hosting:
  npm start  →  http://localhost:3000

Deploy to Cloudflare Pages/Workers:
  wrangler pages deploy  (uses wrangler.json)
